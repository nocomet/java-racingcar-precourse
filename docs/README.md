# 자동차 경주 게임

## 기능 목록
- [ ] 입력부 구현
  - [ ] 경주 할 자동차 이름을 입력 받음
  - [ ] 시도할 회수를 입력 받음
- [ ] 출력부 구현
  - [ ] 각 차수별 실행 결과를 출력
  - [ ] 경기 종료 후 우승자 안내 문구 출력
- [ ] 에러 처리
  - [ ] 사용자 입력에 대한 에러 처리
- [ ] 기능 구현
  - [ ] 입력으로 받은 횟수만큼 동작을 처리
  - [ ] 자동차의 전진 기능 구현 (0~9사이의 값을 랜덤으로 선택하여 4이상일 경우 전진, 3 이하의 값이면 멈춤)
  - [ ] 우승자 판단 로직
