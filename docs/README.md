
# 자동차 경주 게임

## 기능 목록
- [x] 입력부 구현
  - [x] 경주 할 자동차 이름을 입력 받음
  - [x] 시도할 회수를 입력 받음
- [x] 출력부 구현
  - [x] 각 차수별 실행 결과를 출력
  - [x] 경기 종료 후 우승자 안내 문구 출력
- [x] 에러 처리
  - [x] 사용자 입력에 대한 에러 처리
- [x] 기능 구현
  - [x] 입력으로 받은 횟수만큼 동작을 처리
  - [x] 자동차의 전진 기능 구현 (0~9사이의 값을 랜덤으로 선택하여 4이상일 경우 전진, 3 이하의 값이면 멈춤)
  - [x] 우승자 판단 로직
